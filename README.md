# Personal Voice Assistant

## Overview

A Python-based AI Voice Assistant that listens to voice commands and performs tasks such as opening applications, searching the web, and telling the current time.

## Features

- Speech Recognition
- Text to Speech
- Open Google
- Open YouTube
- Open Calculator
- Open Notepad
- Tell Current Time
- Web Search

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- PyAudio

## Run

pip install -r requirements.txt

python assistant.py
